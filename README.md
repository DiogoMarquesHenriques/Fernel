# AGRIX — Site institucional

Site responsivo para a marca de tratores AGRIX (React + Tailwind CSS).

## Requisitos

- Node.js 18+
- npm ou yarn

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) no browser.

## Build

```bash
npm run build
```

O output fica em `dist/`.

## Estrutura do projeto

```
src/
├── components/     # Navbar, Footer, Layout
├── data/           # Dados (ex.: tractors.js)
├── pages/          # Home, Tratores, Sobre Nós, Contacto, TractorDetail
├── App.jsx
├── main.jsx
└── index.css
```

## Páginas

- **Início** — Hero, séries em destaque, pontos fortes, CTA
- **Tratores** — Grelha de modelos (Fernel, FieldMaster, Ironfield)
- **Sobre Nós** — História, Missão, Visão, Valores
- **Contacto** — Formulário, dados da empresa, mapa (placeholder)

Língua: Português (Portugal).
